# Contributing to TradeTracker

Thank you for considering contributing to TradeTracker!

## Stipulation

All code created in Copilot-assisted chats must be duplicated and committed to this GitHub repository to ensure continuity and context.

## Guidelines

- Follow the existing folder structure.
- Use clear and descriptive commit messages.
- Ensure all new features are documented in the README.md file.
- Submit pull requests for review before merging.
